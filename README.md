# Naboo Hydro-GP

**الإصدار:** 1.0.0-stable  
**التقنيات:** HTML5 / Canvas / JavaScript  
**نوع المشروع:** Single Page Application (SPA)  
**لوحة الألوان:** Olive Green / Neon Blue / Neon Yellow

## 🚀 الهدف
لعبة سباق Naboo Hydro-GP تعمل على المتصفح، قابلة للتوسيع وتحويلها إلى PWA.

## ⚙️ التثبيت
1. استنساخ المستودع
2. فتح `index.html` في أي متصفح حديث

## 🧩 الملفات الأساسية
- `/js/main.js` → منطق اللعبة
- `/css/style.css` → التنسيقات
- `/config/game.config.json` → إعدادات اللعبة
- `/checkpoints/checkpoints.json` → نقاط التفتيش
